Privacy Policy for QuickReply Templates

Effective date: March 15, 2026

QuickReply Templates is a Chrome extension that helps users save and insert reusable email templates into Gmail.

Information We Collect

QuickReply Templates stores the templates that users create and save. This information is stored using Chrome extension storage.

How We Use Information

Saved templates are used only to provide the core functionality of the extension, including saving, managing, and inserting templates into Gmail compose windows.

Data Sharing

QuickReply Templates does not sell, rent, or share user data with third parties.

Remote Code

QuickReply Templates does not use remotely hosted code. All executable code is packaged within the extension.

Data Collection

QuickReply Templates does not collect personal information, browsing history outside its intended functionality, or email content for external transmission.

Permissions

The extension requests storage, activeTab, scripting, and Gmail host permissions only to save templates and insert them into Gmail when the user chooses to do so.

Security

We take reasonable steps to keep extension data limited to the functionality described above.

Changes

This privacy policy may be updated from time to time. Updates will be reflected on this page.
